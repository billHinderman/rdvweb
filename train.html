The Gist

The Train Tracker is a project referenced throughout the second part of Building Responsive Data Visualization for the Web. In the book, the project is used to illustrate the following concepts:

  • Progressive enhancement.
  • Building a clean API.
  • Handling AJAX loading.
  • Responsive data visualization.

The actual website can be found at cta.billhinderman.com. That is a Backbone.js project, and its implementation details are obfuscated. However, its design is identical to that in the book, and can be used as a jumping-off point for discussion on design, rather than the canonical reference for implementation details.

The code used and referenced in the book can be found in the github repository.
