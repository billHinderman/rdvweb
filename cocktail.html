The Gist

The Cocktail Guide is a project referenced throughout Building Responsive Data Visualization for the Web. Within the book, the project spans a search results page for a list of cocktail recipes, and the recipe pages themselves.

The search results page is used to illustrate responsive web design, while the recipe page is used to illustrate responsive data visualization within a responsive webpage.

The actual website can be found at thecocktailgui.de, though it is a number of years old and uses readically different coding conventions than those in the book. For design's sake, however, it is identical.

The code used and referenced in the book can be found in the github repository.
